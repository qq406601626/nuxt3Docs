# Output 目录

当构建您的应用程序用于生产时，Nuxt会创建`.output/`目录。

::alert{type=warning}
在运行`nuxt build`时，您不应该触碰其中任何文件，因为整个目录将被重新创建
::

使用此目录将您的Nuxt应用程序部署到生产环境中。在我们的[部署部分](/getting-started/deployment)中了解更多信息。
